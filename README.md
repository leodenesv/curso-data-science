Projetos desenvolvidos durante o curso de Data Science.

## Módulo 5 — Análise de Ações B3

Análise quantitativa da PETR4 usando dados reais.

**Tecnologias:** Python, pandas, yfinance, matplotlib

**O que o projeto faz:**
- Coleta dados reais de cotações via yfinance
- Calcula retorno diário, MM20 e MM50
- Gera gráfico de fechamento com médias móveis
- Produz relatório com métricas de retorno e volatilidade

**Resultado:** PETR4 teve retorno de 66% nos últimos 6 meses
com volatilidade diária de 1.74%.
